*****************
开发工具
*****************

在这个章节中描述的模块将会帮助你编写软件， 例如 :mod:`pydoc` 模块基于使用模块的内容生成文档。
:mod:`doctest` 和 :mod:`unittest` 模块包含编写单元测试的框架， 框架会执行测试代码并验证期望
值，生成结果。  :program:`2to3` 可以将 Python 2.x 的源代码转换成正确的 3.x 的代码。
